# Skill Circle

A simple Flutter starter app with a polished login screen, home screen, and named-route navigation.

## Structure

- `lib/main.dart` boots the app.
- `lib/src/app/app.dart` defines the app theme and routes.
- `lib/src/screens/` contains the login and home screens.
- `lib/src/widgets/brand_panel.dart` holds a reusable branded side panel used by the login screen.

## Getting started

1. Install Flutter.
2. Run `flutter pub get`.
3. Start the app with `flutter run`.
